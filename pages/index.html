ftc-split-deploy

Deploy your FTC team code to the robot in seconds, not minutes.

This site is a Maven repository. In your FtcRobotController project's settings.gradle:

pluginManagement {
    repositories {
        maven { url 'https://repo.cyclezlab.com' }
        gradlePluginPortal()
        google()
        mavenCentral()
    }
}

plugins {
    id 'ftc.splitdeploy' version '0.2.3'
}

Version 0.2 adds base-compatibility checks, transactional split backup and rollback, OnBot Java duplicate detection, configuration-cache-safe tasks, and a non-destructive full-install policy.

Full instructions, troubleshooting, and source: github.com/CycleZLab/ftc-split-deploy
